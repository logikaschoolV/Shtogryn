# LogiTalk
**LogTalk**- це простий і зручний онлайн-чат з підтримкою обміну  повідомлення та зображення в реальному часі.
Побудовано на Pyhon, Tkinter/ i сокета:
## можливості 
Надсилання текстових повідомленнь у реальномуу часі
пітрима кількох користувачів
Авторизація
## використанні технології
Python 3
(CustomTkinter)
socket
Base64
Pillow (PIL)
